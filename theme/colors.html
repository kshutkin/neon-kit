Usage Notes

About This Color System

  • Adaptive Design: All colors automatically adapt between light and dark modes using CSS custom properties.
  • Semantic Naming: Colors are named by their purpose (primary, secondary, success) rather than their appearance (blue, red).
  • Color-mix Technology: Most colors are derived from base colors using the CSS color-mix() function for consistency.
  • OKLCH Color Space: Colors use OKLCH for better perceptual uniformity and color manipulation.
  • Neon Accent System: Vibrant neon colors provide the foundation for bright color palette.
